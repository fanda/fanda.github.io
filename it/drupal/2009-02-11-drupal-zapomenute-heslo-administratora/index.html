11. 2. 2009 Drupal a zapomenuté heslo administrátora

Nemáte-li na svém localhost nakonfigurovaný mail server a stránky v testovacím prostředí Vám zapomenuté heslo nepošlou, máte 2 možnosti. Buď si ten server nakonfigurujete a nebo si heslo změníte jednoduchým SQL příkazem přímo v databázi. Přestože první možnost je z dlouhodobého hlediska jistě výhodnější, vždy volím tu druhou. Její krátký postup se může hodit každému drupalistovi.

Jak v Drupalu změnit zapomnuté heslo admina přístupem do databáze

  1. Přihlašte se do svého databázového systému. S mysql můžete použít phpMyAdmin nebo příkaz "mysql -u user -p".
  2. Přejděte do drupalem používané databáze("use drupal;" v mysql)
  3. Jednoduše spusťte SQL příkaz "UPDATE users SET pass=MD5('heslo') WHERE uid = 1;"

Vše hotovo ve třech krocích.
